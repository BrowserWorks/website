## Verify the releases

```shell
cat $DOWNLOADED_FILE.sha512
# Output of cat command here.

shasum -a 512 $DOWNLOADED_FILE
# Output of shasum command here.

# If the output of both commands is the same, the file is likely to be safe.
```
